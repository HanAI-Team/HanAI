# HanAI 🌿

한의사 전용 AI 진단 보조 서비스

## 서비스 소개
증상 입력 한 번으로 사상체질 판별부터 한약 처방 추천, 자동 차트 작성까지.
한의사 면허 인증을 통해 검증된 한의사만 사용할 수 있는 AI 진단 보조 플랫폼입니다.

## 주요 기능
- 🎙️ 음성 녹음 → 자동 차트 작성 (오토 차팅)
- 🔍 증상 입력 → 사상체질 판별 + 근거 제시
- 💊 한약 처방 추천 (고문헌 출전 포함)
- 📋 환자 기록 자동 저장 및 관리
- 🔬 논문 근거 기반 처방 추천

## 사용 대상
- 한의사 면허번호 인증 후 사용 가능

## 기술 스택
- **Backend**: Python, Flask
- **AI**: Claude Haiku,Clova Speech
- **Database**: SQLite → PostgreSQL
- **Frontend**: HTML / CSS / JavaScript (반응형)

## 데이터
- 한의대 교과서 처방 데이터 2,290건 (신계/간계/심계/폐계/비계내과학)
- 실제 임상 데이터 (한약 처방, 침 처방, 다이어트, 피부미용)
- 지식재산처 한국전통 처방정보 API (신청 중)
