# Zinmac 베타 테스터 가이드

한의원 진료를 돕는 AI 진단 보조 서비스 **Zinmac**의 사용 방법을 안내합니다.
PC, 태블릿, 휴대폰 등 기기 종류에 상관없이 같은 주소로 접속해서 사용할 수 있습니다.

---

## 1. 시작하기

### 1-1. 접속하기
- 주소: https://zinmac.vercel.app
- PC, 태블릿, 휴대폰 브라우저 어디서든 위 주소로 접속하면 됩니다.
- 휴대폰으로 접속하면 화면 하단에 탭(진료/환자/설정)이 나타나는 모바일 화면으로 자동 전환됩니다. (별도 앱 설치 불필요)

### 1-2. 로그인
- **의사(원장)**: 면허번호 + 비밀번호로 로그인
- **직원(간호사/데스크)**: 이메일 + 비밀번호로 로그인
- 로그인 화면 상단 탭에서 "의사" / "직원"을 선택해 전환합니다

### 1-3. 회원가입 (의사 최초 1회)
- 로그인 화면 하단 "회원가입" 클릭
- 이름, 면허번호, 비밀번호, 한의원명을 입력 (주소/전화번호는 선택)
- 가입 후 **관리자 승인 대기** 상태가 됩니다. 승인 완료 안내를 받으면 로그인 가능합니다
- 직원 계정은 원장님이 로그인 후 [설정] 메뉴에서 직접 추가합니다 (2-4 참고)

---

## 2. 사용하기

### 2-1. 홈 화면
- 등록된 환자 수, 오늘 진료 건수, 최근 등록 환자를 한눈에 확인
- 최근 진료 기록 5건 미리보기
- "새 진료 시작" 버튼으로 바로 진료 화면 이동

### 2-2. 진료 — 핵심 기능

좌측에서 **환자를 검색/선택**하거나 "+ 신규 환자 등록"으로 새 환자를 등록한 뒤 시작합니다.

#### ① 진료 기록 탭
다음 방식을 조합해 증상을 입력할 수 있습니다 (최소 하나는 필수).
- **음성 녹음**: 빨간 마이크 버튼을 눌러 진료 대화를 녹음 (다시 누르면 종료)
- **음성 파일 업로드**: 기존 녹음 파일(mp3/wav/m4a/webm, 최대 100MB)을 드래그 앤 드롭
- **증상 직접 입력**: 텍스트로 증상을 직접 작성

음성(녹음/업로드)과 텍스트를 함께 입력하면, 음성을 변환한 내용 뒤에 직접 입력한 텍스트가 추가되어 함께 분석됩니다.

추가로 입력 가능한 항목:
- **메모**: 환자 관련 추가 메모 (환자 프로필에 저장됨)
- **병력**: "있음" 선택 시 상세 병력을 입력하면 AI 분석에 함께 반영됨

모두 입력 후 **"AI 진단 분석 시작"** 버튼을 누르면 분석이 시작됩니다 (음성 또는 텍스트 중 하나는 필수).

#### ② 진단 결과 탭
- AI가 두 가지 결과를 보여줍니다:
  - **결과 1 (데이터셋 기반)**: 임상 사례 데이터를 참고한 진단
  - **결과 2 (AI 종합 소견)**: Claude의 일반적인 진단
- 각 결과는 **사상체질 / 한의학적 진단 + 양방 대응 / 한약 처방 / 침 처방**으로 구성
- 화면 하단의 **"전체 복사"** 버튼으로 EMR(동의보감 등)에 붙여넣기 좋은 형식으로 복사 가능
- ⚠️ **이 결과는 AI 참고용이며, 최종 판단은 담당 한의사가 합니다**

진료가 끝나면:
- **"저장"**: 진료 기록을 환자 차트에 저장 (저장 후 피드백 입력 가능)
- **"인쇄"**: 결과 인쇄
- **"새 진료"**: 새로운 진료 시작 (입력 초기화)

저장 후에는 **피드백(👍/👎 + 의견)** 을 남길 수 있습니다. 베타 테스트 기간에는 결과가 정확했는지/도움이 됐는지 꼭 피드백 부탁드립니다!

#### ③ 진료 이력 탭
- 해당 환자의 과거 진료 기록을 최신순으로 확인
- 각 기록을 펼치면 주소증, 진단 내용, 처방, 메모/병력 확인 가능
- 휴지통 아이콘으로 기록 삭제 가능 (확인 후 삭제)

#### ④ 한의학 검색 탭
- **질문(ask) 모드**: 한의학 관련 궁금한 점을 자유롭게 질문하고 답변 받기
- **진단(diagnose) 모드**: 증상 텍스트를 입력하면 진단 결과(결과1/결과2) 출력
- 진단 결과 아래 **"진료 기록에 저장"** 버튼을 누르면, 결과1/결과2의 사상체질·한의학적 진단·한약 처방·침 처방 중 원하는 항목만 선택해 현재 선택된 환자의 진료 기록에 저장할 수 있습니다 (환자를 먼저 선택해야 함)

### 2-3. 환자 관리
- 환자 이름으로 검색, 목록에서 선택 시 진료 화면으로 이동
- "+ 신규 환자 등록"으로 이름(필수), 성별, 생년월일, 전화번호 입력 후 등록

### 2-4. 설정
- **비밀번호 변경**: 현재 비밀번호 + 새 비밀번호(8자 이상) 입력
- **로그아웃**
- **(원장 전용) 하위 계정 관리**: 직원(간호사/데스크) 계정 추가, 활성화/비활성화, 인원 제한 확인

---

## 3. 베타 테스트 시 꼭 확인해주세요

- AI 진단 결과(사상체질, 처방, 침 처방)가 실제 임상 판단과 비교했을 때 **얼마나 적절한지**
- 음성 녹음/업로드 → 진단까지 **소요 시간**과 **인식 정확도** (특히 한의학 용어)
- "결과 1"과 "결과 2" 중 어느 쪽이 더 유용한지
- 사용 중 **에러 메시지, 끊김, 느린 로딩** 등 발생 여부

버그나 개선 의견은 발견 즉시 캡처/메모해서 알려주시면 바로 확인하겠습니다.

## 4. 자주 묻는 질문 (FAQ)

**Q. 회원가입 후 로그인이 안 돼요.**
A. 의사 계정은 관리자 승인이 필요합니다. 승인 완료 안내를 기다려 주세요.

**Q. 음성 녹음과 텍스트 입력을 동시에 쓸 수 있나요?**
A. 네. 음성을 변환한 내용에 직접 입력한 텍스트가 추가되어 함께 분석됩니다.

**Q. "한의학 검색" 탭에서 진단한 내용도 환자 차트에 남나요?**
A. 기본적으로는 빠른 참고용이라 저장되지 않지만, 결과 아래 "진료 기록에 저장" 버튼을 누르면 원하는 항목만 골라 환자 차트에 저장할 수 있습니다 (환자를 먼저 선택해야 함). 정식 진료는 [진료] 탭에서 진행 후 "저장"을 눌러주세요.
